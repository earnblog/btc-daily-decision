# BTC 每日决策

一个纯静态页面,每天告诉你三件事:**今天做不做、做多还是做空、仓位多大。**

👉 **[打开页面](https://earnblog.github.io/btc-daily-decision/)**

数据实时来自 Binance 公开接口,全部计算在浏览器本地完成,不上传任何东西,无需账号。

---

## 三步规则

| 步骤 | 规则 |
|---|---|
| **1. 今天做不做** | 当前波动率 < 过去一年 25% 分位 → 不开新仓 |
| **2. 方向** | 价格 > 50 日均线 → 只做多；价格 < 50 日均线 → 只做空或空仓 |
| **3. 仓位** | 目标波动率 50% ÷ 当前年化波动率，上限 3 倍 |

**出场**：价格反向穿越 50 日均线时平仓或反手。不设盈利目标。

---

## 为什么是这三条

这三条来自一轮对比测试。被测试并**否决**的指标：

| 指标 | 与 BTC 未来涨跌的关系 | 结论 |
|---|---|---|
| 美联储政策利率 | -0.00 | 无效 |
| 加息/降息事件 | 加息有短期压制，降息无效 | 基本无效 |
| 稳定币总供应量 | +0.02，峰值在 -4 天 | 滞后指标 |
| BTC 现货 ETF 净流入 | +0.03，峰值在 -5 天 | 滞后指标 |
| 永续合约资金费率 | +0.03，峰值在 -3 天 | 滞后指标 |

这些指标与价格的**同期**相关可以很高（ETF 达 +0.76），但交叉相关的峰值全部落在滞后侧 —— 是价格先动，指标跟随。它们是价格的镜子，不是价格的原因。

真正有效的只有两类，且都来自价格自身：

- **趋势/动量** —— 50 日均线策略在 2021-2026 年化 +17.3%，同期买入持有 -2.2%
- **波动率** —— 可预测（相关 +0.57，方向预测仅 +0.03），用于定仓位和过滤

回测结果（同一信号，仅改仓位管理，平均杠杆归一，含手续费与资金费率）：

| 方案 | 年化 | 夏普 | 最大回撤 |
|---|---|---|---|
| 固定仓位 | +17.3% | 0.57 | -62.5% |
| 按波动率调仓 | +25.4% | 0.73 | -53.0% |
| **+ 低波动期空仓** | **+25.4%** | **0.82** | **-42.8%** |

低波动过滤砍掉了 43% 的交易日，收益一分没少，回撤少了 20 个百分点。

## 三条禁令

1. **不跳过信号。** 觉得「这次明显不对」而跳过的那一单，往往就是全年赚钱的那一单。
2. **不因连亏而停。** 连亏 8 次是正常的，不是策略坏了。
3. **不主观加仓。** 仓位只由第三步的公式决定。

---

## 你要接受的代价

- 大部分交易是亏的（胜率约 35~45%），靠少数几笔大的把全年拉回来
- 最大回撤 -40% ~ -60% 量级，这是固有代价，优化不掉
- 每次趋势反转都会还持着仓，确认了才出，必然吃到那一段

## 参数

| 参数 | 值 | 说明 |
|---|---|---|
| 均线周期 | 50 日 | 20~200 都有效，按想抓的周期选，别回测优化 |
| 目标波动率 | 50% | 按能承受的回撤定 |
| 静默分位 | 25% | 可试 20~35%，别精调 |
| 杠杆上限 | 3 倍 | 建议更保守 |

参数曲面应该是平缓高原，不是一根针。只有某个特定数字有效 = 过拟合。

改参数直接编辑 `index.html` 顶部的常量：

```js
const MA_DAYS = 50, TARGET_VOL = 0.50, QUIET_PCT = 0.25, MAX_LEV = 3.0;
```
